有时候会记录一些随手笔记，因此开辟了这个板块！

 

  1. #1.5.7版本mysql解压版安装问题

解压版不再包含data目录，需要初始化：
bin\mysqld –defaults-file=my.ini –initialize-insecure
bin\mysqld –install

  1. #2.java连接mysql数据库小实例

package com.test;

import java.sql.;
import com.mysql.jdbc.;
public class test
{
public static void main(String[] args) throws Exception,SQLException
{
Class.forName(“com.mysql.jdbc.Driver”);
java.sql.Connection conn = DriverManager.getConnection(“jdbc:mysql://localhost:3306/mysqlname?user=root&password=123456”);
java.sql.Statement sql=conn.createStatement();
java.sql.ResultSet rs=sql.executeQuery(“select * from tablename;”);
while(rs.next())
{
System.out.println(rs.getString(1));
}
}
}
